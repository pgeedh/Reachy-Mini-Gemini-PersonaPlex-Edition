# Changelog

## [0.1.0] - 2026-01-23

### Added
- **Empath Eye**: Face detection and emotion recognition using `fer`.
- **Empath Brain**: Integration with Gemma-2b for LLM capabilities.
- **Empath Voice**: Text-to-Speech support.
- **Dashboard**: Next.js based glassmorphic UI.
- **Simulation**: Support for MuJoCo simulation.

### Changed
- Replaced "Sentinel" architecture with "Empath" architecture.
- Updated project goals to focus on HRI and emotional connection.
